## 2024-03-20 - Localization in Accessibility Attributes
**Learning:** The application's UI is localized in Portuguese. When adding UX/accessibility improvements (e.g., `aria-label` or `title` attributes), it is essential to ensure the text is in Portuguese (e.g., 'Fechar' for close actions, 'Mostrar/Ocultar senha' for password toggles) to maintain consistency and provide an inclusive experience for the primary user base.
**Action:** Always verify the application's primary language by inspecting existing text elements and use appropriate translations for all accessibility labels and tooltips.
